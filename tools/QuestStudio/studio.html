Chapitres de l'histoire

Publication

Enregistrer écrit le fichier du jeu sur ce PC. Publier l'envoie sur GitHub — le serveur du jeu fait ensuite git pull + redémarrage et tous les joueurs voient les nouveautés.

Édition

Bienvenue

Choisis une quête à gauche ou crée-en une nouvelle. Chaque quête dit au joueur quel monstre tuer, combien de fois, et ce qu'il gagne en revenant voir le PNJ.

Relie les quêtes entre elles avec « Quête suivante » pour construire une histoire en chapitres — la suivante ne se débloque que quand la précédente est rendue.

Survole n'importe quel objet pour voir sa fiche (stats, valeur), comme en jeu. Les récompenses se choisissent avec leur icône, pas besoin de retenir les IDs.
